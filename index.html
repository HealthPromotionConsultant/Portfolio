How I work

Nonprofits need sessions that are engaging, safe, and easy to run—especially with time pressure, interpreter needs, and mixed literacy levels. I use plain language, interactive activities, and differentiated instruction to support learning.

Facilitation principles

  • Trauma-informed: choice, safety, and strengths-based language built into the session structure
  • Culturally safe: humility, respect, and participant voice—aware of power dynamics
  • Evidence-based: clear, science-informed explanations without jargon
  • Practical + accessible: adaptable pacing, interpreter-friendly delivery, and tools participants can start using the same day

What organizations receive

  • Session agenda + slide deck
  • Plain-language handouts + resources (print + digital)
  • Optional facilitator notes, mixed-literacy engagement plan, and pre/post reflection prompts (interpreter-friendly)
  • Optional short feedback form + attendance and top-themes snapshot
