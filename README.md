# AnnK Multi-Page Portfolio Site (Home / Workshops / About / Contact)

## What’s included
- index.html (Home)
- workshops.html
- about.html
- contact.html (mailto-based inquiry form — no backend needed)
- styles.css + script.js

## Quick setup (fastest)
### Option A: Netlify Drop
1. Unzip the folder
2. Go to Netlify → “Deploy manually” / “Drop” (wording varies)
3. Drag the entire folder contents into the upload area
4. You’ll get a live link you can share immediately

### Option B: GitHub Pages
1. Create a repo
2. Upload all files to the repo root
3. Settings → Pages → Deploy from branch (main)
4. Your site goes live at the Pages URL

## Customize in 10 minutes
1. Open `contact.html` and set your email:
   - Find: data-to="your@email.com"
   - Replace with your real email

2. Replace the download placeholders:
   - Put your PDFs into /assets
   - Rename them to:
     - Capability_Statement.pdf
     - Full_Portfolio.pdf
   - Or edit the links in index.html

3. Update your links:
   - Add your LinkedIn / website anywhere you want in the footer or contact page.

## Optional upgrades (high ROI)
- Add 1–2 testimonials on Home (Selected experience section)
- Add a “Results Snapshot” line (attendance range, satisfaction score, repeat bookings)
- Add a simple booking link (Calendly) in Contact page
